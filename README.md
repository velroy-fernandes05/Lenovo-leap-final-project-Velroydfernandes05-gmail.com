# AI Study Assistant

## Lenovo LEAP AI Capstone Project

### SDG Goal

SDG 4 – Quality Education

### Problem Statement

Students often struggle to find personalized learning support, quick answers to academic questions, and organized educational resources.

### Solution

AI Study Assistant is a web-based platform that helps students through:

* AI Chat Tutor
* Learning Dashboard
* Educational Resources
* Quiz System
* Student Support Features

### Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub

### Future Scope

* OpenAI API Integration
* Voice Assistant
* Mobile Application
* Cloud Database
* Multi-language Support
